# Car Sales Power BI Dashboard

## Project Overview

This project presents an interactive Power BI dashboard designed to analyze car sales performance across multiple dealer regions in the United States.

The dashboard provides insights into:

* YTD Total Sales
* Average Car Price
* Cars Sold
* Weekly Sales Trends
* Sales by Dealer Region
* Sales by Body Style
* Sales by Color

## Tools & Technologies

* Power BI
* DAX
* Power Query
* Data Modeling
* Data Visualization

## Dashboard Features

* Interactive filters and slicers
* KPI cards
* Geographical sales analysis
* Trend analysis
* Dynamic visualizations
* Performance tracking dashboards

## Business Insights

* Identified top-performing dealer regions
* Analyzed sales distribution by body style and color
* Tracked weekly sales fluctuations
* Compared company-wise sales performance
