# Hospitality Revenue Analytics Dashboard – Power BI Project

## Project Overview

The Hospitality Revenue Analytics Dashboard is an interactive Power BI solution designed to analyze hotel performance metrics, occupancy trends, revenue generation, customer realization rates, and booking platform effectiveness.

This dashboard enables hospitality businesses to monitor operational KPIs, evaluate room performance, analyze customer booking behavior, and improve strategic decision-making through data-driven insights.

---

## Objectives

* Monitor hotel revenue and occupancy performance
* Analyze room category utilization and realization rates
* Track weekly and city-wise occupancy trends
* Evaluate booking platform contribution and customer behavior
* Identify revenue trends across business and luxury segments
* Support hospitality performance optimization and operational planning

# Dashboard Modules

## 1. Hospitality Performance Overview Dashboard

### KPI Metrics

* Revenue
* RevPAR (Revenue Per Available Room)
* DSRN (Daily Sellable Room Nights)
* Occupancy %
* ADR (Average Daily Rate)
* Realisation %

### Occupancy Analysis

* Occupancy % by category
* Occupancy % by week number
* Weekly trend monitoring

### Property Performance Analysis

* Revenue by property
* RevPAR analysis
* Occupancy comparison
* DSRN, DBRN, and DURN tracking
* Cancellation analysis

### Booking Platform Analysis

* Realisation % by booking platform
* Customer acquisition trend analysis
* Platform performance comparison

### Interactive Filters

* City-based filtering
* Room type filtering
* Weekly timeline selection
