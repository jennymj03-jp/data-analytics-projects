# Bank Loan Analytics Dashboard – Power BI Project

## Project Overview

The Bank Loan Analytics Dashboard is a comprehensive Power BI solution developed to analyze loan application trends, funded amounts, repayment performance, customer segmentation, and lending risk metrics.

This interactive dashboard provides financial institutions and banking stakeholders with actionable insights into loan portfolio performance, customer behavior, loan purposes, and repayment trends to support strategic lending and credit risk management decisions.

---

## Objectives

* Monitor overall loan application performance
* Analyze funded and received loan amounts
* Evaluate good and bad loan distributions
* Identify customer borrowing patterns
* Track loan trends across states and time periods
* Support data-driven lending and risk management decisions

# Dashboard Modules

## 1. Loan Summary Dashboard

### KPI Metrics

* Total Loan Applications
* Total Funded Amount
* Total Amount Received
* Average Interest Rate
* Average Debt-to-Income (DTI) Ratio
* Month-over-Month (MoM) analysis

### Good vs Bad Loan Analysis

#### Good Loan Metrics

* Good Loan Issued Percentage
* Good Loan Applications
* Good Funded Amount
* Good Amount Received

#### Bad Loan Metrics

* Bad Loan Issued Percentage
* Bad Loan Applications
* Bad Funded Amount
* Bad Amount Received

### Loan Status Performance

Analysis based on:

* Fully Paid Loans
* Current Loans
* Charged-Off Loans

### Interactive Filters

* Loan Grade
* Loan Purpose
* Customer Segmentation

### Financial Comparison Reporting

* Application Trends
* Funded Amount Comparison
* Amount Received Analysis
* Interest Rate Comparison
* DTI Comparison
